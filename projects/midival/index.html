MIDIVal

MIDIVal #

MIDIVal is a TypeScript library to help you manage your MIDI messages with ease. It allows an easy-to-use, event-based syntax. It aims to be compatible with the Browser, Node.js API, and, in the future, React Native.

Features #

  • Easy-to-use event-based syntax
  • Compatible with Browser and Node.js API
  • Future support for React Native
